【画像フォルダについて】

このフォルダに画像ファイルを保存してください。

■ ロゴ画像
  images/logo.png  または  images/logo.png
  → ヘッダーに表示されます（推奨サイズ: 高さ 40〜60px 程度）

■ メニュー商品画像
  images/menu/plain.png         → プレーンチュロス
  images/menu/chocolate.png     → チョコレートチュロス
  images/menu/cinnamon.png      → シナモンチュロス
  images/menu/milk-butter.png   → ミルクバターチュロス
  images/menu/set3.png          → チュロス3種セット
  images/menu/set2.png          → チュロス2種セット
  images/menu/drink-set.png     → チュロスドリンクセット
  images/menu/ice.png           → アイスチュロス

  推奨サイズ: 横400px × 縦300px 以上（正方形が望ましい）
  ファイル形式: jpg / png どちらでも可

※ 画像ファイルを配置するまでは、代わりにカラー背景が表示されます。
